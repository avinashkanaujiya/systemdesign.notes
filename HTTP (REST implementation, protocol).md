http is an API that has four methods - read, delete, update.

Passing queries within the URL is ok, but passing data via encoding it in URL is not, not a [[REST]]ful way.

Verbs are 
- GET 
- POST 
- PUT 
- [[PATCH (HTML verb)]] 
- DELETE
